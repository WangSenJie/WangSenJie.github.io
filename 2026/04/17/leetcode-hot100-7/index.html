滑动窗口

$\star$(3) 无重复字符的最长子串

给定一个字符串 s, 请你找出其中不含有重复字符的最长子串的长度.

使用滑动窗口法和Hash表, 用 l 和 r 表示滑动窗口的左右边界, hash 表中存储当前滑动窗口中每个字符的下标的上确界

  • 初始化 l = 0, r = 0, hashtable = {}, max_length = 0
  • 右指针 r 慢慢向右滑动
    • 若 s[r] 在 hashtable 中, 且 hashtable[s[r]] >= l, 则 r 位置元素与窗口中的元素有重复, 此时我们要找到与之重复的元素的下标, 即 hashtable[s[r]]. 更新 l <- hashtable[s[r]] + 1, hashtable[s[r]] <- r
    • 更新 max_length <- max(max_length, r-l+1)

(438) 找字符串中所有字母异位词

给定两个字符串 s 和 p, 找到 s 中所有 p 的 异位词的子串, 返回这些子串的起始索引. 不考虑答案输出的顺序.

采用滑动窗口和计数器 Counter()

  • 初始化 p 的计数器 p_count <- Counter(p), s 子串的计数器 s_count <- Counter(s[:len(p)-1]) (初始时刻 s_count 的长度为 len(p)-1 ), 初始化滑动窗口左端点 left <- 0
  • 从下标 len(p)-1 开始遍历 s 字符串, 记当前下标为 right (滑动窗口的右端点)
    • 将当前遍历到的 s[right] 加入到 s_count 中
    • 判断 s_count = p_count 是否成立. 若相等, 则记录此时窗口的起始索引 right-len(p)+1;
    • 将窗口的左端点往右移动
