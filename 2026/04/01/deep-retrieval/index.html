发表于 2026-04-01 更新于 2026-04-02 分类于 推荐算法
Deep Retrieval的原理
  • 经典的双塔模型把用户、物品表征成向量, 线上做最近邻查找.
  • Deep Retrieval 把物品表征成路径(path), 线上查找用户最匹配的路径.
  • Deep Retrieval 类似于阿里的 TDM

索引

  • 物品到路径的索引: item -> list <path>
    • 一个物品可以对应多条路径
    • 用 个节点表示一条路径
  • 路径到物品的索引: path -> list <item>
    • 一条路径对应多个物品
